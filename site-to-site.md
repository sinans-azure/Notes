# Site to Site

The site to site connection is used to connect one site(On-cloud) to another site(On-premise).

- This feature is used mostly when you do some computation in a Cloud Server and some in the On-Premise server but treat both as one.

1. Create a resource group.
2. Create a virtual network.
3. Create a gateway subnet under it.
4. Create a local network gateway.
5. Create a virtual network gateway.
6. Create a connection between the virtual network gateway and the local network gateway.
7. Verify the connection.
